You can make the following GET requests

  • /file/sample.html -- returns the content of the file sample.html
  • /json -- returns a json of the /random request
  • /random -- returns index.html/li>

You can also make the following queries

  • /base-convert?

    Converts a given number from base x to base y.
    Parameters:
    • num : Input number to be converted
    • from : Base to convert from
    • to : Base to convert into
    Example of use:
    /base-convert?num=23&from=10&to=2

  • /weather?

    Gets current weather forecast for place specified by coordinates.
    Supports both plain coordinates but also cardinal ones.
    Parameters:
    • lat : Latitude input
    • lon : Longitude input
    Examples of use:
    /weather?lat=39.7456&lon=-97.0892 :: (Linn, KS)
    /weather?lat=47.6061N&lon=122.3328W :: (Seattle, WA)

  • /multiply?

    Multiplies two numbers.
    Parameters:
    • num1 : first number
    • num2 : second number
    Example of use:
    /multiply?num1=5&num2=3

  • /github?

    Finds the given user and displays information about their public GitHub repositories.
    Parameters:
    • query : the url information for the given user, formatted as "users/{USERNAME}/repos"
    Example of use:
    /github?query=users/amehlhase316/repos
